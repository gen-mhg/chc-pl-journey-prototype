How the PL application journey branches on login once Approval Odds goes live

Once Approval Odds — which requires login — is added to the personal loan listing, logged-in members get a streamlined, credit-card-style flow that takes them straight to "My Rewards" to claim; logged-out users follow the existing flow — navigating back manually, hunting for the right reward form, and filling in every field by hand. This demo focuses on the contrast between the two journeys.

How to use: Use the toggle in the black DEMO bar at the top to switch between "Logged in ↔ Logged out". Each switch resets to the listing page and runs a full journey in that state. Click "Journey comparison" at any time to see a step-count breakdown.

Logged-in member Streamlined (CC-style) flow

  • Approval Odds (High / Medium) shown directly on listing cards
  • After applying, auto-redirected with a clear path back to MoneyHero
  • Lands directly on "My Rewards" showing application progress and a claim card
  • Name / email / phone pre-filled and locked
  • Only needs to enter ARN + select reward
  • CHC membership eligibility already satisfied — no extra steps

Logged-out user Existing flow

  • Approval Odds locked: 🔒 Log in to view your approval chances
  • After applying, lands on provider page with no return button
  • Must navigate back to the listing page and scroll to find the reward section
  • Must hunt for the WeLend form in a long list of reward forms
  • All 6 fields must be filled in manually
  • All fields must be filled in manually — system does not recognise the user

Journey comparison: Logged in vs Logged out

Same reward, two very different journeys. Login (Approval Odds) saves members several steps — no hunting, auto pre-fill, and a direct path to "My Rewards" to complete the claim.

Logged-in member

5 key steps
  1. See Approval Odds on the listing → Apply Now
  2. Apply on WeLend's site (external)
  3. Receive a MoneyHero reward email (EDM) → log in
  4. "My Rewards" — application progress + claim card
  5. Form pre-filled; enter ARN + pick reward → done
Guided by email, correct form surfaced in "My Rewards", contact details pre-filled Because the member is logged in, MoneyHero recognises them — no hunting, and name / email / phone are auto-filled and locked.

Logged-out user

6 key steps
  1. Approval Odds locked → Apply Now
  2. Sent straight to the provider page (no return button)
  3. Manually navigate back to the MoneyHero listing
  4. Scroll down to find the reward section → Fill form
  5. Hunt for the WeLend form in a long list
  6. Fill all fields by hand — system does not recognise the user → done
Manual navigation back, hunt for the right form, all fields filled by hand with no pre-fill System does not recognise the user — all details must be re-entered; high drop-off risk.
